Data Projects

This page has links to my open-source data projects. The code for these projects is available on my Github page.

The Social Web of Doctor Who

This project used D3.js to create a social network graph showing connections between the major characters of the British television series Doctor Who.

The State of Major League Pitching: A Visual Investigation

This project visualizes pitching statistics for Major League baseball, using both tradition metrics such as ERA and WHIP to more advanced metrics such as xFIP and WAR. Historical trends and the rise of incidents in Tommy John surgery are also investigated.

2018 Atlantic Hurricanes

This project allows the user to view an interactive map of the 2018 Atlantic hurricane season, including commentary.

Annual Box Office Analysis (2018)

This project visualizes the 365 days of 2018 at the US movie box office, analyzing trends by MPAA rating, film distributor and time of film release.

Gender Politics

This project looks at the uneven division of unpaid labor by gender for many countries around the world.
